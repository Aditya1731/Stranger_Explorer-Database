# Stranger Explorer (SQL Project)

A trip companion matching database system.

## Features
- Users create trips
- Others join trips (many-to-many)
- Messaging inside trips
- Reviews after completing trip
- Matching system between travelers

## Tech Stack
- MySQL Database
- Pure SQL: DDL + DML + Joins
